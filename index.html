Leçon sur Hugging Face Spaces

Présentation générale

Hugging Face est une plateforme spécialisée dans l’intelligence artificielle qui permet de partager des modèles d’IA, publier des datasets, collaborer sur des projets et déployer des applications intelligentes. Elle fonctionne avec un système de versionnement similaire à GitHub. Une partie importante de cette plateforme est appelée Spaces.

Qu’est-ce que Hugging Face Spaces

Hugging Face Spaces permet de créer, héberger et partager des applications web basées sur l’intelligence artificielle. Un Space est donc une application web contenant un modèle d’IA et accessible via un lien internet. On peut y trouver des chatbots, des générateurs d’images, des traducteurs automatiques, des résumés de texte ou encore des outils d’analyse de sentiments.

Architecture d’un projet Spaces

Un projet Spaces contient généralement plusieurs fichiers : app.py qui est le programme principal, requirements.txt qui liste toutes les bibliothèques nécessaires comme Gradio, Transformers ou PyTorch, README.md pour documenter le projet et un dossier model qui contient le modèle IA ou le relie à un modèle en ligne.

Technologies utilisées

Les technologies utilisées permettent de créer l’application et de faire fonctionner l’intelligence artificielle. Gradio est le framework qui crée l’interface utilisateur. Il permet de créer des champs de saisie, d’afficher les résultats et de connecter l’utilisateur au modèle IA, sans coder en HTML ou JavaScript. Streamlit est un autre framework utilisé pour créer des applications interactives orientées data science, par exemple des dashboards ou des visualisations de données. Docker permet de déployer une application complète dans un environnement personnalisé, utile lorsque l’application nécessite plusieurs services, langages ou configurations serveur. Transformers est la bibliothèque qui permet de charger et d’utiliser facilement des modèles pré-entraînés, et PyTorch est le framework qui réalise les calculs nécessaires aux modèles IA.

Fonctionnement combiné

Dans une application Spaces, Gradio récupère les entrées de l’utilisateur, Transformers envoie ces données au modèle et PyTorch réalise les calculs. Le résultat est ensuite affiché dans l’interface Gradio.
